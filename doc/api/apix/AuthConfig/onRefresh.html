onRefresh property

RefreshCallback? onRefresh
final

Callback to refresh tokens when a refresh-triggering status code is received.

This is the legacy approach. Consider using refreshEndpoint with onTokenRefreshed for a simpler flow.

If both onRefresh and refreshEndpoint are null, no automatic refresh will be attempted. If both are provided, refreshEndpoint takes priority.
