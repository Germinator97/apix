Future<Result<T, ApiException>> getResult()

Executes this future and wraps the result in a Result.

If the future completes successfully, returns Success with the value. If the future throws an ApiException, returns Failure with the error. Other exceptions are rethrown.

Implementation

Future<Result<T, ApiException>> getResult() async {
  try {
    return Result.success(await this);
  } on ApiException catch (e) {
    return Result.failure(e);
  }
}
