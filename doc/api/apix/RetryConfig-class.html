RetryConfig

RetryConfig class

Configuration for retry behavior.

Defines how failed requests should be retried, including the maximum number of attempts and which status codes trigger retry.

Example:

final config = RetryConfig(
  maxAttempts: 3,
  retryStatusCodes: [500, 502, 503, 504],
);
