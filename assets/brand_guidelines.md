# ApiX Brand Guidelines

## Logo

L'icône représente une **étoile à 4 branches** symbolisant :
- Point de connexion API
- Performance (spark/étincelle)
- Les 4 piliers : Auth, Retry, Cache, Observability

## Couleurs

### Palette Principale

| Nom | Hex | RGB | Usage |
|-----|-----|-----|-------|
| **Deep Navy** | `#1E3A5F` | 30, 58, 95 | Fond cercle principal |
| **Border Blue** | `#3D5A80` | 61, 90, 128 | Bordure cercle |
| **Spark Orange** | `#F89C35` | 248, 156, 53 | Étoile, accent |
| **Slate Gray** | `#4A5568` | 74, 85, 104 | Texte secondaire |
| **Light Gray** | `#F7FAFC` | 247, 250, 252 | Fond clair |

### Mode Sombre

| Nom | Hex | Usage |
|-----|-----|-------|
| **Orange** | `#FF6B35` | Fond icône |
| **White** | `#FFFFFF` | Étoile, texte |
| **Dark BG** | `#0D1117` | Arrière-plan |

## Typographie

- **Logo** : System UI / SF Pro / Inter (weight 600)
- **Headers** : Inter Bold
- **Body** : Inter Regular

## Fichiers Logo

```
assets/
├── logo_primary.svg    # Logo principal (fond clair)
├── logo_dark.svg       # Logo pour fond sombre
├── logo_icon.svg       # Icône seule (favicon, app icon)
```

## Utilisation

### ✅ Do
- Conserver les proportions
- Espace minimum autour du logo = hauteur de l'icône
- Utiliser sur fonds unis

### ❌ Don't
- Déformer le logo
- Changer les couleurs arbitrairement
- Placer sur images complexes
